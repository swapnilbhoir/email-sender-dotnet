
>IDE : Microsoft Visual Studio Professional 2013

> Project Description :

This project deals with sending email using different email API.
In case one api fails to send email then another email api is used so the user experience remains intact.

> Email Api used : 1.MailGun  2.sparkpost

> Project Flow:

User enters the receipient email id ,subject and body then click on send email button.

Then appropriate message regarding mail is displayed .
